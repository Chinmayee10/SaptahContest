# Problem Statement
Given an integer array arr[], find the contiguous subarray which has the largest sum and return its sum.
N= no. of elements, k= no. of elements that make up the maximum sum  
  
# Example:  

## Input:
              1 4 2 10 23 3 1 0 20
              4
## Output: 
              39 {4+2+10+23}
